# 02: ボールを動かそう

このしょうでは、ボールをつくって自動で動かし、壁にぶつかったら跳ね返るようにします。

:::message
**このチャプターで学ぶこと**
- 座標（x, y）と速度（dx, dy）の関係
- ボールが自動で動く仕組み
- 壁で「跳ね返る」反射の実装方法
- 符号を反転させる考え方（`-self.dx`）
:::

## プログラムのもくてき

- 「ボール」をつくって、自動で動かす
- 壁（左・右・上）にぶつかったら跳ね返る
- 下に落ちたら、中央からやり直す（仮）

## コードのポイント

### 1. ボールの設定

```python
# ボールの設定
BALL_SIZE = 12
```

- ボールの大きさを12ピクセルに設定しています
- 定数にしておくと、後で大きさを変えたいときに便利です

### 2. ボールクラスをつくる

```python
class Ball(pygame.sprite.Sprite):
    """動き回るボール"""

    def __init__(self):
        super().__init__()
        self.image = pygame.Surface((BALL_SIZE, BALL_SIZE))
        self.image.fill(WHITE)
        self.rect = self.image.get_rect()
        # 画面の中央に配置
        self.rect.centerx = SCREEN_WIDTH // 2
        self.rect.centery = SCREEN_HEIGHT // 2
        # ボールの速度（dx: 横方向, dy: 縦方向）
        self.dx = 4
        self.dy = -4
```

- `self.dx` は横方向の速度です（正なら右、負なら左に動く）
- `self.dy` は縦方向の速度です（正なら下、負なら上に動く）
- `dy = -4` なので、最初は上に向かって動きます

:::details 💡 self（セルフ）ってなに？
`self` は **「自分自身」** という意味です。
ゲームの中には「ボール」がたくさん出てくるかもしれません。
そのとき、「このボールの速度」なのか「あのボールの速度」なのかを区別するために、
「**自分（self）** の速度（dx）」という書き方をします。
:::

:::message alert
**速度と座標の関係**
- `dx = 4` のとき、毎フレーム x座標が 4 増える → 右に動く
- `dx = -4` のとき、毎フレーム x座標が 4 減る → 左に動く
- `dy = -4` のとき、毎フレーム y座標が 4 減る → 上に動く（画面では上が y=0）
:::

### 3. ボールを動かす

```python
def update(self):
    """ボールを動かして、壁で跳ね返す"""
    # ボールを移動
    self.rect.x += self.dx
    self.rect.y += self.dy
```

- 毎フレーム、ボールの位置に速度を足しています
- これだけで、ボールが自動的に動き続けます

### 4. 壁で跳ね返る

```python
    # 左右の壁で跳ね返る
    if self.rect.left <= 0:
        self.rect.left = 0
        self.dx = -self.dx  # 横方向の速度を反転
    if self.rect.right >= SCREEN_WIDTH:
        self.rect.right = SCREEN_WIDTH
        self.dx = -self.dx

    # 上の壁で跳ね返る
    if self.rect.top <= 0:
        self.rect.top = 0
        self.dy = -self.dy  # 縦方向の速度を反転
```

- `self.dx = -self.dx` で、速度の符号を反転しています
- 例：`dx = 4`（右に動く）→ `-4`（左に動く）
- これが「跳ね返り」の仕組みです

### 5. 下に落ちたらリセット

```python
    # 下に落ちたら上から再スタート（仮）
    if self.rect.top >= SCREEN_HEIGHT:
        self.rect.centerx = SCREEN_WIDTH // 2
        self.rect.centery = SCREEN_HEIGHT // 2
        self.dy = -4
```

- ボールが画面の下に落ちたら、中央に戻します
- 次のチャプターで、これを「ゲームオーバー」に変更します

## 最終的なコード

```python
import pygame
import sys

# 画面サイズ
SCREEN_WIDTH = 480
SCREEN_HEIGHT = 640

# 色の定義
BLACK = (0, 0, 0)
WHITE = (255, 255, 255)
BLUE = (0, 128, 255)

# パドルの設定
PADDLE_WIDTH = 80
PADDLE_HEIGHT = 15

# ボールの設定
BALL_SIZE = 12


class Paddle(pygame.sprite.Sprite):
    """プレイヤーが操作するパドル（板）"""

    def __init__(self):
        super().__init__()
        self.image = pygame.Surface((PADDLE_WIDTH, PADDLE_HEIGHT))
        self.image.fill(BLUE)
        self.rect = self.image.get_rect()
        self.rect.centerx = SCREEN_WIDTH // 2
        self.rect.bottom = SCREEN_HEIGHT - 30
        self.speed = 8

    def update(self):
        """キー入力でパドルを左右に動かす"""
        keys = pygame.key.get_pressed()

        if keys[pygame.K_LEFT]:
            self.rect.x -= self.speed
        if keys[pygame.K_RIGHT]:
            self.rect.x += self.speed

        if self.rect.left < 0:
            self.rect.left = 0
        if self.rect.right > SCREEN_WIDTH:
            self.rect.right = SCREEN_WIDTH


class Ball(pygame.sprite.Sprite):
    """動き回るボール"""

    def __init__(self):
        super().__init__()
        self.image = pygame.Surface((BALL_SIZE, BALL_SIZE))
        self.image.fill(WHITE)
        self.rect = self.image.get_rect()
        # 画面の中央に配置
        self.rect.centerx = SCREEN_WIDTH // 2
        self.rect.centery = SCREEN_HEIGHT // 2
        # ボールの速度（dx: 横方向, dy: 縦方向）
        self.dx = 4
        self.dy = -4

    def update(self):
        """ボールを動かして、壁で跳ね返す"""
        # ボールを移動
        self.rect.x += self.dx
        self.rect.y += self.dy

        # 左右の壁で跳ね返る
        if self.rect.left <= 0:
            self.rect.left = 0
            self.dx = -self.dx  # 横方向の速度を反転
        if self.rect.right >= SCREEN_WIDTH:
            self.rect.right = SCREEN_WIDTH
            self.dx = -self.dx

        # 上の壁で跳ね返る
        if self.rect.top <= 0:
            self.rect.top = 0
            self.dy = -self.dy  # 縦方向の速度を反転

        # 下に落ちたら上から再スタート（仮）
        if self.rect.top >= SCREEN_HEIGHT:
            self.rect.centerx = SCREEN_WIDTH // 2
            self.rect.centery = SCREEN_HEIGHT // 2
            self.dy = -4


def main():
    pygame.init()
    screen = pygame.display.set_mode((SCREEN_WIDTH, SCREEN_HEIGHT))
    pygame.display.set_caption("ブロック崩し - 02")

    # パドルとボールを作成
    paddle = Paddle()
    ball = Ball()
    paddle_group = pygame.sprite.Group(paddle)
    ball_group = pygame.sprite.Group(ball)

    clock = pygame.time.Clock()

    while True:
        clock.tick(60)

        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                pygame.quit()
                sys.exit()

        # 更新
        paddle_group.update()
        ball_group.update()

        # 描画
        screen.fill(BLACK)
        paddle_group.draw(screen)
        ball_group.draw(screen)

        pygame.display.flip()


if __name__ == "__main__":
    main()
```

このチャプターでは、ボールを動かして壁で跳ね返す仕組みを学びました。

- **速度（dx, dy）** を使って、ボールを自動で動かしました
- **符号の反転** で、壁での「跳ね返り」を実装しました
