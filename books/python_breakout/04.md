# 04: ブロックを並べよう

このしょうでは、ブロックを画面に並べて、ボールが当たったらブロックが消える仕組みを作ります。

:::message
**このチャプターで学ぶこと**
- リスト（配列）を使って複数のデータを管理する方法
- 二重ループ（for文の中のfor文）で格子状にオブジェクトを配置
- `pygame.sprite.Group` を使った複数オブジェクトの管理
- `spritecollide()` 関数で複数オブジェクトとの衝突判定
:::

## プログラムのもくてき

- ブロックを画面に並べる
- ボールがブロックに当たったら、ブロックが消える
- ボールはブロックで跳ね返る

![第4章の実行結果](/images/python_breakout/chapter04.png)

## コードのポイント

### 1. 色のリストを作る

```python
# ブロックの色リスト
BLOCK_COLORS = [
    (255, 100, 100),  # 赤
    (255, 200, 100),  # オレンジ
    (255, 255, 100),  # 黄色
    (100, 255, 100),  # 緑
    (100, 200, 255),  # 水色
]
```

- リスト（配列）を使って、複数の色を管理しています
- `BLOCK_COLORS[0]` で赤、`BLOCK_COLORS[1]` でオレンジを取り出せます
- 行ごとに色を変えるために使います

### 2. ブロックの設定

```python
# ブロックの設定
BLOCK_WIDTH = 58
BLOCK_HEIGHT = 20
BLOCK_ROWS = 5      # 縦に5行
BLOCK_COLS = 8      # 横に8列
BLOCK_MARGIN = 2    # ブロック間の隙間
BLOCK_TOP_MARGIN = 60  # 上からの余白
```

- ブロックの大きさや配置に関する設定をまとめています
- 定数にしておくと、後で調整するときに便利です

### 3. ブロッククラス

```python
class Block(pygame.sprite.Sprite):
    """壊されるブロック"""

    def __init__(self, x, y, color):
        super().__init__()
        self.image = pygame.Surface((BLOCK_WIDTH, BLOCK_HEIGHT))
        self.image.fill(color)
        self.rect = self.image.get_rect()
        self.rect.x = x
        self.rect.y = y
```

- ブロックは `x`, `y`, `color` を受け取って作成します
- 各ブロックが異なる位置と色を持てるようになっています

### 4. ブロックを並べる関数

```python
def create_blocks():
    """ブロックを並べて作成する"""
    blocks = pygame.sprite.Group()

    for row in range(BLOCK_ROWS):
        for col in range(BLOCK_COLS):
            # ブロックの位置を計算
            x = col * (BLOCK_WIDTH + BLOCK_MARGIN) + BLOCK_MARGIN
            y = row * (BLOCK_HEIGHT + BLOCK_MARGIN) + BLOCK_TOP_MARGIN

            # 行ごとに色を変える
            color = BLOCK_COLORS[row % len(BLOCK_COLORS)]

            block = Block(x, y, color)
            blocks.add(block)

    return blocks
```

- **二重ループ** で、行と列を組み合わせてブロックを配置しています
- `range(5)` は 0, 1, 2, 3, 4 の5つの数を順番に返します
- `col * (BLOCK_WIDTH + BLOCK_MARGIN)` で、ブロックの横位置を計算しています

:::message alert
**% 演算子（余りの計算）**

`row % len(BLOCK_COLORS)` は「行番号を色の数で割った余り」です。

- row=0: 0 % 5 = 0 → 赤
- row=1: 1 % 5 = 1 → オレンジ
- row=2: 2 % 5 = 2 → 黄色
- row=5: 5 % 5 = 0 → 赤（また最初に戻る）

これで色を繰り返し使えます。
:::

### 5. ブロックとの衝突判定

```python
# ブロックとの衝突
hit_blocks = pygame.sprite.spritecollide(ball, block_group, True)
if hit_blocks:
    ball.dy = -ball.dy
```

- `spritecollide()` は、1つのスプライトとグループ全体の衝突を判定します
- 第3引数の `True` は「衝突したスプライトを削除する」という意味です
- 戻り値 `hit_blocks` は、衝突したブロックのリストです
- 衝突したブロックがあれば、ボールの縦方向を反転させます

### 6. リスタート時のブロック再生成

```python
if event.key == pygame.K_r and game_over:
    ball.reset()
    block_group = create_blocks()  # ブロックを再生成
    game_over = False
```

- リスタート時は、ブロックも新しく作り直します

## 最終的なコード

```python
import pygame
import sys

# 画面サイズ
SCREEN_WIDTH = 480
SCREEN_HEIGHT = 640

# 色の定義
BLACK = (0, 0, 0)
WHITE = (255, 255, 255)
BLUE = (0, 128, 255)
RED = (255, 100, 100)

# ブロックの色リスト
BLOCK_COLORS = [
    (255, 100, 100),  # 赤
    (255, 200, 100),  # オレンジ
    (255, 255, 100),  # 黄色
    (100, 255, 100),  # 緑
    (100, 200, 255),  # 水色
]

# パドルの設定
PADDLE_WIDTH = 80
PADDLE_HEIGHT = 15

# ボールの設定
BALL_SIZE = 12

# ブロックの設定
BLOCK_WIDTH = 58
BLOCK_HEIGHT = 20
BLOCK_ROWS = 5
BLOCK_COLS = 8
BLOCK_MARGIN = 2
BLOCK_TOP_MARGIN = 60


class Paddle(pygame.sprite.Sprite):
    """プレイヤーが操作するパドル（板）"""

    def __init__(self):
        super().__init__()
        self.image = pygame.Surface((PADDLE_WIDTH, PADDLE_HEIGHT))
        self.image.fill(BLUE)
        self.rect = self.image.get_rect()
        self.rect.centerx = SCREEN_WIDTH // 2
        self.rect.bottom = SCREEN_HEIGHT - 30
        self.speed = 8

    def update(self):
        keys = pygame.key.get_pressed()
        if keys[pygame.K_LEFT]:
            self.rect.x -= self.speed
        if keys[pygame.K_RIGHT]:
            self.rect.x += self.speed
        if self.rect.left < 0:
            self.rect.left = 0
        if self.rect.right > SCREEN_WIDTH:
            self.rect.right = SCREEN_WIDTH


class Ball(pygame.sprite.Sprite):
    """動き回るボール"""

    def __init__(self):
        super().__init__()
        self.image = pygame.Surface((BALL_SIZE, BALL_SIZE))
        self.image.fill(WHITE)
        self.rect = self.image.get_rect()
        self.reset()

    def reset(self):
        self.rect.centerx = SCREEN_WIDTH // 2
        self.rect.centery = SCREEN_HEIGHT // 2
        self.dx = 4
        self.dy = -4

    def update(self):
        self.rect.x += self.dx
        self.rect.y += self.dy

        if self.rect.left <= 0:
            self.rect.left = 0
            self.dx = -self.dx
        if self.rect.right >= SCREEN_WIDTH:
            self.rect.right = SCREEN_WIDTH
            self.dx = -self.dx
        if self.rect.top <= 0:
            self.rect.top = 0
            self.dy = -self.dy

    def is_out(self):
        return self.rect.top >= SCREEN_HEIGHT


class Block(pygame.sprite.Sprite):
    """壊されるブロック"""

    def __init__(self, x, y, color):
        super().__init__()
        self.image = pygame.Surface((BLOCK_WIDTH, BLOCK_HEIGHT))
        self.image.fill(color)
        self.rect = self.image.get_rect()
        self.rect.x = x
        self.rect.y = y


def create_blocks():
    """ブロックを並べて作成する"""
    blocks = pygame.sprite.Group()

    for row in range(BLOCK_ROWS):
        for col in range(BLOCK_COLS):
            # ブロックの位置を計算
            x = col * (BLOCK_WIDTH + BLOCK_MARGIN) + BLOCK_MARGIN
            y = row * (BLOCK_HEIGHT + BLOCK_MARGIN) + BLOCK_TOP_MARGIN

            # 行ごとに色を変える
            color = BLOCK_COLORS[row % len(BLOCK_COLORS)]

            block = Block(x, y, color)
            blocks.add(block)

    return blocks


def main():
    pygame.init()
    screen = pygame.display.set_mode((SCREEN_WIDTH, SCREEN_HEIGHT))
    pygame.display.set_caption("ブロック崩し - 04")

    font = pygame.font.Font(None, 48)

    # ゲームオブジェクトを作成
    paddle = Paddle()
    ball = Ball()
    paddle_group = pygame.sprite.Group(paddle)
    ball_group = pygame.sprite.Group(ball)
    block_group = create_blocks()

    clock = pygame.time.Clock()
    game_over = False

    while True:
        clock.tick(60)

        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                pygame.quit()
                sys.exit()
            if event.type == pygame.KEYDOWN:
                if event.key == pygame.K_r and game_over:
                    ball.reset()
                    block_group = create_blocks()
                    game_over = False

        if not game_over:
            paddle_group.update()
            ball_group.update()

            # パドルとの衝突
            if ball.rect.colliderect(paddle.rect):
                ball.rect.bottom = paddle.rect.top
                ball.dy = -abs(ball.dy)

            # ブロックとの衝突
            hit_blocks = pygame.sprite.spritecollide(ball, block_group, True)
            if hit_blocks:
                ball.dy = -ball.dy

            if ball.is_out():
                game_over = True

        # 描画
        screen.fill(BLACK)
        block_group.draw(screen)
        paddle_group.draw(screen)
        ball_group.draw(screen)

        if game_over:
            text = font.render("GAME OVER", True, RED)
            text_rect = text.get_rect(center=(SCREEN_WIDTH // 2, SCREEN_HEIGHT // 2))
            screen.blit(text, text_rect)

            restart_text = font.render("Press R to Restart", True, WHITE)
            restart_rect = restart_text.get_rect(center=(SCREEN_WIDTH // 2, SCREEN_HEIGHT // 2 + 50))
            screen.blit(restart_text, restart_rect)

        pygame.display.flip()


if __name__ == "__main__":
    main()
```

- **spritecollide()** で、複数のブロックとの衝突を一度に判定しました
